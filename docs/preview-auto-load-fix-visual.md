## Key Improvements Summary

| Aspect                        | Before                    | After                     |
| ----------------------------- | ------------------------- | ------------------------- |
| **Preview on empty project**  | Tries to load immediately | Waits for code            |
| **Preview during generation** | May start prematurely     | Waits until complete      |
| **Chat code visibility**      | All code blocks shown     | Hidden with placeholder   |
| **User feedback**             | Confusing states          | Clear status at each step |
| **Timing**                    | Immediate (500ms)         | Smart delay (1000ms)      |
| **Error handling**            | Preview fails on no files | Proper state management   |
