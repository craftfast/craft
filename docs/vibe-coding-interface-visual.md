## Color Scheme

### Light Mode

- Background: `#fafaf9` (neutral-50)
- Surface: `#ffffff` (white)
- Border: `#e7e5e4` (neutral-200)
- Text: `#262626` (neutral-800)
- Muted: `#78716c` (neutral-500)

### Dark Mode

- Background: `#171717` (neutral-950)
- Surface: `#262626` (neutral-900)
- Border: `#404040` (neutral-700)
- Text: `#e5e5e5` (neutral-200)
- Muted: `#a3a3a3` (neutral-400)

## Interactive Elements

All buttons and inputs use:

- `rounded-full` for small buttons and inputs
- `rounded-2xl` for cards and panels
- `rounded-xl` for dropdowns and modals
- Smooth transitions: `transition-colors duration-200`
- Hover states with subtle background changes
