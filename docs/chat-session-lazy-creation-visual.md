### Database Checks

```sql
-- Should return 0 empty sessions
SELECT * FROM chat_sessions cs
WHERE NOT EXISTS (
  SELECT 1 FROM chat_messages cm
  WHERE cm.chatSessionId = cs.id
);

-- All sessions should have at least 1 message
SELECT cs.id, cs.name, COUNT(cm.id) as message_count
FROM chat_sessions cs
LEFT JOIN chat_messages cm ON cm.chatSessionId = cs.id
GROUP BY cs.id, cs.name
HAVING COUNT(cm.id) = 0;
```
