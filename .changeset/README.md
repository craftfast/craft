# Changesets

This folder manages version changes for the Craft project.

## Quick Usage

After making changes, run:

```bash
pnpm changeset
```

Then select the change type (patch/minor/major) and describe your changes.

For detailed documentation, see [docs/versioning-guide.md](../docs/versioning-guide.md).

## Files in this folder

- `config.json` - Changeset configuration
- `*.md` files - Pending changesets (auto-generated, commit these!)
